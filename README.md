A Frontend Mentor e-commerce product page with GSAP animations, lightbox gallery, interactive cart, and user panel.

## Features

- GSAP-powered entrance and scroll animations
- Lightbox image gallery with thumbnail navigation
- Interactive cart with add/remove functionality
- User account panel
- Fully responsive layout

## Built with

- HTML5
- CSS3
- JavaScript
- GSAP 3
